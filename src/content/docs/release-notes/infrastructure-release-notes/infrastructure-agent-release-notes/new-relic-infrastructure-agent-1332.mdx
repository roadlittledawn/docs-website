A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend you upgrade the infrastructure agent regularly, at least every three months. As of this release, the oldest supported version is [agent version 1.20.7](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1207).

## Fixed

* Cache NTP value for the specified interval [#1488](https://github.com/newrelic/infrastructure-agent/pull/1488)
* NTP timeout in seconds [#1488](https://github.com/newrelic/infrastructure-agent/pull/1488)
