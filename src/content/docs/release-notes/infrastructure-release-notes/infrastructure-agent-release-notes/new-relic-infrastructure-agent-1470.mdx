A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
We recommend you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [infrastructure agent 1.28.4](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1284).

## Added

* Packages for Debian 12 [#1724](https://github.com/newrelic/infrastructure-agent/pull/1724)
* Container decoration with Docker and Containerd dual sockets [#1725](https://github.com/newrelic/infrastructure-agent/pull/1725)

## Changed

* Bump `newrelic/nri-docker` dependency to v1.8.16 [#1727](https://github.com/newrelic/infrastructure-agent/pull/1727)
